# Agent 开发学习路线（Python 版，从易到难）

## ⚠️ 前提条件

本教程**只关注 Agent Loop 的实现逻辑**，不关注模型配置。

开始前请确保已完成以下任一配置：

- ✅ **本地部署**：vLLM / Ollama 等本地模型服务
- ✅ **远程账号**：OpenAI 或其他兼容 OpenAI API 的远程服务

### 📄 配置文件

项目根目录的 `config.json` 用于统一管理模型配置：

```json
{
  "model": {
    "provider": "vllm",
    "name": "qwen3.5-122b",
    "base_url": "http://36.134.71.54:8000",
    "max_tokens": 120000
  }
}
```

### 🧪 测试模型配置

先运行测试脚本，确认模型配置正常：

```bash
python test_model.py
```

> ✅ **开始学习前，请确保测试通过！**

## 🥇 第 0 阶段：Hello World（单次调用）

### 🎯 目标
先搞清楚最基础的一件事：
> LLM = 一个函数

### 🧩 你要实现
输入一句话 → 调用模型 → 输出结果

### 核心点
- API 调用（OpenAI / vLLM / 本地模型）
- prompt 输入输出

### ✅ 结果形态
```
Q: 你好
A: 你好！有什么可以帮你？
```

---

## 🥈 第 1 阶段：带上下文的对话（Chat Memory）

### 🎯 目标
让模型"记住你刚说过什么"

### 🧩 你要实现
多轮对话 = history 拼接

### 核心点
- messages 列表
- role（user / assistant / system）
- token 控制

### ✅ 结果
```
用户：我叫张三
用户：我叫什么？
模型：你叫张三
```

> 👉 到这里，你只是一个"聊天机器人"，还不是 agent

---

## 🥉 第 2 阶段：手动工具调用（非 Agent）

### 🎯 目标
让模型"间接"使用工具（但不是自动）

### 🧩 你要实现
用户问天气 → 你手动调用天气 API → 再喂给模型

### 核心点
- Tool = Python 函数
- 你自己决定什么时候调用

### 🚨 本质
还不是 agent，因为模型不会自己决定

> 👉 到这里，你学会了"工具是什么"和"如何手动调用"

---

## 🏅 第 3 阶段：结构化输出（Tool Calling 前置）

### 🎯 目标
让模型输出"可解析结构"

### 🧩 你要实现
让模型输出 JSON：
```json
{
  "action": "search",
  "input": "北京天气"
}
```

### 核心点
- 输出解析（parser）
- JSON 校验
- prompt 约束

---

## 🧠 第 4 阶段：单步 Agent（One-shot Tool Use）

### 🎯 目标
模型自己决定是否调用工具（但只调用一次）

### 🧩 你要实现
用户输入 → LLM 判断 → 是否调用 tool → 返回结果

### 核心点
- tool 描述（非常关键）
- tool schema
- 判断分支

### ✅ 示例流程
```
Q: 北京天气？
→ LLM: 调用 weather_tool
→ 执行 tool
→ 返回答案
```

---

## 🔁 第 5 阶段：Agent Loop（核心突破）

### 🎯 目标
实现真正的：
> 思考 → 行动 → 观察 → 再思考

### 🧩 你要实现
循环：
```python
while not done:
    1. LLM 思考（Thought）
    2. 输出 Action
    3. 执行 Tool
    4. 写入 Observation
```

### 核心点
- loop 控制
- max_steps（防死循环）
- 中间状态记录

### 🚨 这是分水岭
> 👉 到这里，你已经是 OpenClaw 类 agent 了

---

## 📋 第 6 阶段：模型做计划（Planning）

### 🎯 目标
让模型学会"先思考再行动"，制定详细的执行计划

### 🧩 你要实现
用户提出复杂任务 → LLM 分解为多个子任务 → 按计划逐步执行

### 核心点
- 任务分解（Task Decomposition）
- 计划生成与修正
- 子任务优先级排序
- 反思与迭代

### ✅ 示例流程
```
Q: 帮我写一篇关于人工智能的文章
→ LLM: 计划步骤：
   1. 搜索 AI 的定义和发展历史
   2. 整理 AI 的主要应用领域
   3. 分析 AI 的未来趋势
   4. 撰写完整文章
→ 执行步骤1 → 执行步骤2 → 执行步骤3 → 执行步骤4 → 返回最终结果
```

### 关键技术
- Prompt Engineering 引导结构化思考
- 计划模板（Plan Template）
- 自我反思机制

> 👉 到这里，你的 agent 具备了"远见"和"策略思维"
